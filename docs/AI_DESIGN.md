# Empire of Minds — AI Design

## AI Vision

Empire of Minds should eventually feel like it has smarter, more strategic opponents than typical 4X games.

This does not mean starting with an LLM-controlled game.

The initial AI should be deterministic, debuggable, and rule-based. More advanced AI can be introduced later behind stable interfaces.

## Core AI Principle

AI chooses from legal actions generated by the game engine.

The game engine is the judge.

The AI must never:
- mutate state directly
- bypass the validator
- invent actions outside the action schema
- execute arbitrary code or commands
- rely on hidden information unless explicitly allowed by difficulty design

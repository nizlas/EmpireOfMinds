# Prototype assets (Empire of Minds)

These are **prototype-only**, **non-final** assets.

- **Generated** or **placeheld** art is allowed for **internal testing**.
- Assets are **replaceable**; swap files without expecting gameplay to break.
- **No gameplay system** may depend on **exact pixel content** of these files.
- **Prompts** and **provenance** should be retained **when practical** (see per-folder `PROVENANCE.md` where present).
- **Lore**, **final aesthetics**, **IP / licensing review**, and **release-asset** decisions are deferred to **Phase 6**.
